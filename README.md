\# Tweet Emotion Recognition



This project classifies emotions in tweets using deep learning.



\## Emotions

\- Sadness

\- Love

\- Joy

\- Surprise

\- Anger

\- Fear



\## Model

\- Tokenizer + Padding

\- Embedding Layer

\- Bidirectional LSTM

\- Softmax Output



\## Tech Stack

\- Python

\- TensorFlow / Keras

\- NumPy

\- Matplotlib
